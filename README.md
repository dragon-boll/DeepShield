# 🛡️ DeepShield: Android Remote Monitor
Sistema de monitoreo y streaming en tiempo real para dispositivos Android utilizando **Termux** y **Shizuku**.

## 🚀 Funcionalidades
- Streaming de video en tiempo real vía web.
- Ejecución de comandos con privilegios de Shell (UID 2000).
- Automatización mediante Bash scripts.

## 🛠️ Tecnologías
- Bash, Python, Shizuku, ADB.
